# Project Folder Generator

A Python CLI tool to automatically generate standard project folder structures with pre-configured files.

## Features

- Quick initialization with a single command
- Three templates: standard, Python, and web
- Auto-generates README.md, .gitignore, and requirements.txt
- No external dependencies required

## Installation

```bash
git clone <repository-url>
cd project-folder-generator
```

## Usage

```bash
python main.py my-project
python main.py my-project -t python
python main.py my-project -t web
```

See USAGE.md for more details.

## Requirements

Python 3.7+
